# 交互演示版(沙盒试用入口)

> 无需部署即可在浏览器体验产品全流程;数据仅存访问者浏览器 localStorage,与真实应用无关。

- **在线入口**(Claude Artifact,仓库所有者账号私有,可从页面分享):
  https://claude.ai/code/artifact/903d0db0-6509-4c29-9fd6-9967280e7bb8
- **源文件**:[`docs/demo/记账本演示.html`](demo/记账本演示.html)(自包含单文件,可直接双击本地打开)

## 覆盖的功能(与真实应用对齐)

| 模块 | 演示内容 |
|------|----------|
| 登录/注册 | 门禁逻辑同真实应用;演示账号只存本地 |
| 首页 | hero 卡合并月份切换/结余/支出/收入;单行提醒条(到期=琥珀、超支=红,首项+剩余计数);按日流水 |
| 编辑账目 | 点条目弹**底部编辑面板**:改金额/分类,**独立红色删除键**(确认后删) |
| 记一笔 | 支出/收入分段切换;常规小数键盘(含 `.` 与 ⌫)+ 分类宫格 + 备注 |
| OCR 导入 | 模拟识别 → 待确认清单(存疑标黄默认不勾)→ 勾选入账 |
| 基金 | 总市值、较上期涨跌;「+ 手动记」快照表单与「📷 OCR」持仓截图模拟识别(无代码项标疑默认不勾、同代码归一、同日覆盖);详情页跨期涨跌表 + 持有期累计 + 逐条删除 |
| 消费分析 | 环比、近 6 月趋势、分类结构、预算执行进度 |
| 预算设置 | 分类限额,清空即取消 |
| 周期支出 | 表单新增、到期标签、记一笔顺延、流水候选一键采纳 |
| 我的 | 用户名展示、**中/EN 语言切换**(全页面生效,偏好独立保存)、退出登录、重置演示数据 |

## 与真实应用的差异

- 中英切换为演示版特性(真实应用暂无 i18n);中文为源语言,EN 词典缺词回退中文,基金名等专有名词不译。
- 数据在 localStorage,无服务端;OCR 为固定样例;基金详情暂无走势图(真实应用有 SVG 走势)。
- 认证为演示模拟(明文存本地),真实应用为 scrypt + HttpOnly 会话 + proxy 中央守卫(ADR 0012)。

## 维护约定

- 真实应用新增用户可见功能后,**同步更新本演示**(改 `docs/demo/记账本演示.html`,再重新发布到上面同一 Artifact 链接)并刷新本文件的功能表。
- **设计令牌**(2026-07 简化版重构):细描边扁平卡片(`--line` 1px,无投影,弹层除外);主色浅 `#2f7a68`/深 `#5cb39d`;语义色仅用于提醒(琥珀=到期、红=超支/删除);金额统一 `tabular-nums`,大数 30px/750;二级页统一「‹ 标题 + 右侧动作」头部;提醒一律单行条(首项 + 剩余计数),不占大块。
- 改动后必跑冒烟脚本(注册→提醒→编辑面板→记一笔→OCR→基金→分析→预算→周期支出→登出)且浅/深两主题截图目检。
