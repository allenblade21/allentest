# 安卓端设置指南 — 手机随时随地记账

> 服务器已按 [部署.md](部署.md) 跑在家里的 MacBook 上(Tailscale 内网,ADR 0011)。
> 本文是**安卓手机侧**的配置步骤;iPhone 步骤类似(App Store 装 Tailscale + Safari 加主屏)。

## 一、装 Tailscale 并加入你的网络(一次性,约 3 分钟)

1. 安装 Tailscale 安卓版:
   - 有 Google Play:直接搜索 **Tailscale** 安装;
   - 无 Google Play(国产 ROM):到 https://tailscale.com/download/android 下载官方 APK 安装(或 F-Droid)。
2. 打开 App → **Log in**,用与 MacBook 端**同一个账号**登录(Google/GitHub/微软账号均可)。
3. 首次连接会请求建立 VPN 连接,点**允许**;顶部开关保持 **Connected**。
4. 在 App 的设备列表里应能看到你的 MacBook(如 `macbook-pro`)。记下它的**机器名或 100.x.x.x IP**。

> Tailscale 是点对点加密内网,不是翻墙代理;流量只在你自己的设备之间走,个人版免费。

## 二、打开应用并添加到主屏幕

1. 手机浏览器(推荐 Chrome)访问:
   ```
   http://<MacBook机器名>:3000     # 例:http://macbook-pro:3000
   http://100.x.x.x:3000           # 机器名不通就用 Tailscale IP
   ```
2. 首次会跳到登录页 → 点「去注册」创建自己的账号(注册完成后建议按 [部署.md](部署.md) 在服务器端设 `ALLOW_REGISTER=0` 关闭注册)。
3. **加到主屏幕当 App 用**:Chrome 右上 **⋮ → 添加到主屏幕**(部分国产浏览器叫「发送到桌面」)→ 确认。之后从桌面图标一点直达,全屏体验接近原生 App。

## 三、日常使用要点

- **随时随地可用**:只要手机 Tailscale 处于 Connected、家里 MacBook 醒着,4G/5G/任何 WiFi 下都能访问;
- **省电**:Tailscale 空闲时几乎不耗电,可常开;若手机厂商省电策略杀后台,把 Tailscale 加入「不受限制」名单;
- **同一 WiFi 的备选**:不想开 Tailscale 时,在家可直接访问 `http://<MacBook局域网IP>:3000`(出门即失效);
- **拍账单直接导入**:首页「⤓ 导入」→ 拍照/相册选账单截图 → 待确认清单 → 入账。

## 四、故障排查

| 现象 | 检查 |
|------|------|
| 打不开页面 | 手机 Tailscale 是否 Connected;MacBook 是否休眠(合盖会断,见部署.md 防休眠);Mac 端 `launchctl list \| grep jizhangben` |
| 机器名访问不通 | 改用 100.x.x.x IP;或在 Tailscale 管理台开启 MagicDNS |
| 提示注册已关闭 | 服务器端 `.env.local` 设了 `ALLOW_REGISTER=0`,请让管理员临时改为注释并重启服务 |
| 图标打开是白屏 | 浏览器缓存了旧版本,下拉刷新;升级服务后首次访问同理 |

## 附录:备选方案 —— 用闲置安卓手机当服务器(Termux)

> 适合没有常开电脑、但有一台闲置安卓机的场景。**当前项目采用的仍是 MacBook 方案**,此附录仅为备选参考,未做长期验证。

1. 安装 [Termux](https://f-droid.org/packages/com.termux/)(建议 F-Droid 版),然后:
   ```bash
   pkg update && pkg install nodejs-lts git python build-essential
   git clone https://github.com/allenblade21/allentest.git && cd allentest
   npm run setup && npm run build
   npm start        # 监听 3000 端口
   ```
2. 该手机同样安装 Tailscale 并登录同一账号,其余设备按本文第二节访问它。
3. 注意事项:
   - `better-sqlite3` 需要本地编译,Termux 下偶有失败,重跑 `npm install` 或先 `pkg install clang` 再试;
   - 安卓的电池优化会杀后台:Termux 需 acquire wakelock(通知栏按钮)并加入省电白名单;
   - 长期供电别用劣质充电器;数据备份同样用 `npm run backup`(建议 cron 由 Termux:Boot 触发);
   - 稳定性不如 MacBook/树莓派方案,重要数据务必定期把 `backups/` 拷出。
