# ADR 0003 — OCR 用视觉大模型而非传统 OCR

状态:✅ 生效 · 2026-07

## 背景

需要把支付宝/微信/银行截图、小票识别成带字段的账目(金额、商户、时间、分类)。传统 OCR(百度/腾讯/PaddleOCR)只输出文字行,版式多变时需为每种版式写解析规则,且平台改版即失效。

## 决策

用**视觉大模型**直接把截图转成结构化 JSON,并顺带做分类推测。不采用传统 OCR。

## 后果

- ✅ 识别 + 字段结构化 + 分类推测一步到位,无需维护解析规则。
- ✅ 中文界面理解好;结构化输出可用 JSON Schema / json_object 约束。
- ⚠️ 按量付费(个人量级几元/月);依赖外部 API。
- 兜底:所有结果进「待确认清单」,人工确认后才入账(见 ADR 0004)。
