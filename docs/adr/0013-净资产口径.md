# ADR 0013 · 净资产口径:现金结余 + 基金最新市值

- 状态:已采纳(2026-07-19)
- 关联:P3 迭代 3「净资产总览 + 月度报告」;`src/lib/report.ts`、`/networth`、`/report`;TC-NW / TC-RP

## 决策

1. **净资产 = 现金结余 + 基金市值**。现金结余 = 账户初始余额合计 + 全部历史(收入 − 支出);基金市值 = 各基金**最新一期快照**合计(与投资总览同口径)。transfer 不影响总额(账户间移动)。
2. **不引入负债/固定资产等科目**:个人记账场景两类资产已覆盖绝大多数需求,科目体系是过度设计;将来需要再加表。
3. **月度报告是聚合视图,不落表**:实时从流水/预算算出(结余、支出环比、分类 Top5、最大单笔 Top3、预算执行),无新数据模型;月份可切换回看。

## 备选与否决理由

- 按账户逐一记余额并对账:需要用户维护每个账户当前余额,违背「简洁易用」;初始余额 + 流水推导已够。
- 报告快照落库(月末生成):增加任务调度与陈旧数据问题;个人数据量实时算毫无压力。

## 影响

- 无新表;新增 lib/report.ts 纯函数(cashBalanceCents / netWorth / monthlyReport),TC-NW 2 例 + TC-RP 3 例。
- 「我的」新增两个入口;「敬请期待」仅剩异常支出提醒。
