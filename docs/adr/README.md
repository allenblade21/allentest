# 决策记录(ADR)

Architecture Decision Records —— 记录**关键决策及其原因**,是项目的长期记忆。

## 规则

- 每条决策一个文件:`000N-简短标题.md`,**编号递增、内容不可变**(改主意就新增一条并标注取代关系,不改旧的)。
- 格式:**背景 → 决策 → 后果**,简明即可。
- 何时写:做了影响后续、且非显而易见的技术/产品选择时。
