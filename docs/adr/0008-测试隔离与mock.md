# ADR 0008 — 测试用独立库 + OCR mock + 手机视口

状态:✅ 生效 · 2026-07

## 背景

端到端测试需要可复现、零成本、不污染开发数据,且 OCR 依赖外部 API 不能在 CI 里真调。

## 决策

- **独立测试库**:`data/test.db`,在 Playwright `webServer` 启动前(`tests/setup-db.ts`)重建 + 迁移 + seed;不碰开发用的 `data/app.db`。
- **OCR mock**:测试服务器以 `OCR_MOCK=1` 启动,识别返回固定样例(3 正常 + 1 存疑),零 API 成本、结果可断言。
- **手机视口**:390×840(iPhone 尺寸),符合手机优先定位。
- 用例登记表 `docs/测试用例.md`,TC 编号 ↔ spec 标题;串行执行(共享测试库)。

## 后果

- ✅ 测试随时零成本复现,不依赖真实 Key。
- ✅ 每用例自动截图、失败留 trace,报告可查。
- 教训:`webServer` 早于 Playwright `globalSetup` 启动,建库必须放在 `webServer.command` 里而非 globalSetup(见 memory/lessons.md)。
